Create a GitHub Account

If you don't have a GitHub account yet, create one first:

  • Go to github.com/signup

  • Create a password and pick a username (like firstname-lastname)

Install GitHub CLI

Download and install the GitHub CLI:

  • Mac: Open Terminal and run: brew install gh
  • Windows: Download from cli.github.com

Login & Clone the Skills

Open Terminal and run these commands:

Follow the prompts to login with your browser. Then clone the skills:

gh repo clone TrueLegacyHomes/Skills2.0 ~/.tlh-skills
5

Run the Install Script

This creates the slash commands:
