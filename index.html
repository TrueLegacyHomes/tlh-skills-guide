Get Started in 2 Minutes

TLH Skills for Claude Code

Add True Legacy Homes AI skills to your Claude Code in three simple steps.

1

Open Claude Code

Launch Claude Code on your computer. If you don't have it yet, download it from claude.ai/code

2

Add the TLH Marketplace

In Claude Code, type this command and press Enter:

/plugin marketplace add TrueLegacyHomes/tlh-cowork-plugins
3

Pick a Skill & Start Working

Click the skill picker at the top of Claude Code, select a TLH skill, and start chatting. That's it!
