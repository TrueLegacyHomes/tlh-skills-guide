Create a GitHub Account

If you don't have a GitHub account yet, create one first:

  • Go to github.com/signup
  • Enter your email address (use one you check regularly)
  • Create a password and pick a username (like firstname-lastname)

3

Create a GitHub Token

Go to github.com/settings/tokens/new and create a token:

  • Note (name): Claude Code
  • Expiration: 90 days (or longer)
  • Scope: Check the box for repo (Full control of private repositories)
  • Click Generate token
  • Copy the token immediately â€” you won't see it again!
4

Open Terminal & Add Your Token

Open Terminal:

  • Mac: Press Cmd + Space, type Terminal, press Enter
  • Windows: Press Windows key, type PowerShell, press Enter

Paste this command (replace paste_your_token_here with your actual token):

echo 'export GITHUB_TOKEN="paste_your_token_here"' >> ~/.zshrc

Then run this to apply the change:

source ~/.zshrc

Open Claude Code

Add the TLH Skills
